title New note process

Browser->Server: HTTP POST https://studies.cs.helsinki.fi/exampleapp/new_note
Server->Browser: HTTP Response :status code 302: location: /exampleapp/notes

note over Server:
An HTTP response with a status code of 302 causes a
redirect to the new URL specified in the header field Location
end note

Browser->Server:HTTP Get https://studies.cs.helsinki.fi/exampleapp/notes
Server-->Browser: HTML-code
Browser->Server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.css
Server-->Browser: main.css
Browser->Server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.js
Server-->Browser: main.js

note over Browser:
Browser begins executing main.js.
The code contains a request to the server for JSON data
end note
Browser-->Server: HTTP GET https://studies.cs.helsinki.fi/exampleapp/data.json
Server --> Browser : JSON data is 

note over Browser
An event handler is now executed by the browser to display the returned JSON data ie the notes.
end note
